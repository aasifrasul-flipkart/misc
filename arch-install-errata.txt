If using syslinux for the bootloader:
syslinux can't use ext4 with the 64bit feature, which is now the default in mkfs.ext4.
Instead, create without the 64bit feature:
$ mkfs.ext4 -O \^64bit /dev/sda1

Use fs labels instead of uuids:
$ e2label /dev/sda1 <label>
$ genfstab -L /mnt >> /mnt/etc/fstab

In /boot/syslinux/syslinux.cfg use:
    APPEND root=LABEL=ROOT rw

No ethernet on first start, so setup netctl:
$ cd /etc/netctl
Create a new profile from the example:
$ cp examples/ethernet-dhcp .
Get interface name with:
$ ip link
Update 'Interface' and uncomment 'IP6=stateless':
$ vim ethernet-dhcp
Enable and start profile:
$ netctl enable ethernet-dhcp
$ netctl start ethernet-dhcp

Create a user:
useradd -m -G wheel <username>
(-m creates ~)

Sudo:
$ pacman -S sudo
$ vim /etc/sudoers:
  Uncomment %wheel entry

GUI:
$ pacman -S xfce4
$ pacman -S xterm (not installed by default!)
$ echo 'exec startxfce4' > ~/.xinitrc
$ startx (when desired)

lockscreen:
$ pacman -S slock

no boot menu:
Comment out both "UI" directives in /boot/syslinux/syslinux.cfg

disable terminal beep:
$ echo 'set bell-style none' > ~/.inputrc
$ bash (to refresh)
